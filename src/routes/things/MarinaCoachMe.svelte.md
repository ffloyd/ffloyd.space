My first [SvelteKit](https://svelte.dev/) project.
I did a coaching services website for my girlfriend.

I could use some static site generator or similar solution.
But I decided to do a less boring thing.

SvelteKit can be used as a foundation for (mostly) static website.
I was curious to try it for the task.

I'd say it's slower to start, but cheaper in the long run.
Especially if you want custom design and structure.

I liked the process so much that I reused the approach for the site you're currently on.
